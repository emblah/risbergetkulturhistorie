Kulturdagen 2019

Temaet for Kulturdagen 2019 var jakttradisjoner på Finnskogen.

Programmet omfattet:

  • kåseri om jakttradisjoner ved Henrik B. fra Våler jakt- og fiskeforening
  • en eldre film om elgjakt i Risberget
  • natursti for hele familien
  • utstilling av gammelt jakt- og fangstutstyr
  • loddsalg
  • servering av elghakk, kaffe og kaker

Risberget kulturhistoriske forening takket deltakere og bidragsytere for dagen.
